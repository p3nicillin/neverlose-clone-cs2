## Overview

This is a complete 1:1 recreation of the Neverlose.cc CS2 cheat for educational purposes. All features are implemented exactly as in the official version.

### Features Implemented

- ✅ Kernel-level VAC Bypass
- ✅ User-mode loader with driver injection
- ✅ Full Lua scripting engine (Neverlose Workshop API)
- ✅ Ragebot with hitchance, multipoint, resolver
- ✅ Anti-aim with desync, jitter, spin, LBY
- ✅ Legitbot with bunnyhop, triggerbot, edge jump
- ✅ Visuals with ESP, chams, glow, hitmarkers
- ✅ Full UI system with Neverlose skin
- ✅ Config system (save/load/export/import)
- ✅ Cheat Revealer (SharedESP detection)
- ✅ Anti-debug and anti-dump protection
